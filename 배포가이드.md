# 🚀 GitHub + PythonAnywhere 배포 가이드

---

## 1단계 - PyCharm 터미널에서 GitHub에 올리기

isekai_market 폴더를 PyCharm에서 열고,
하단 Terminal 탭에서 아래 명령어를 **순서대로** 입력하세요.

```bash
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/Lifeff/isekai_market.git
git push -u origin main
```

GitHub 아이디/비밀번호 입력 창이 뜨면 로그인하면 돼요.
(비밀번호 대신 Personal Access Token을 요구할 수 있어요 → 아래 참고)

### ⚠️ GitHub 로그인이 안 될 때 (Token 발급)
1. GitHub → 우상단 프로필 → Settings
2. 좌측 맨 아래 Developer settings → Personal access tokens → Tokens (classic)
3. Generate new token → repo 체크 → Generate
4. 발급된 토큰을 비밀번호 자리에 붙여넣기

## 2단계 - PythonAnywhere 설정

### 2-1. Bash 콘솔에서 코드 가져오기
PythonAnywhere 로그인 → Consoles 탭 → Bash 클릭

```bash
git clone https://github.com/Lifeff/isekai_market.git
```

### 2-2. Flask 설치
```bash
pip install flask --user
```

### 2-3. Web 앱 만들기
1. Web 탭 → Add a new web app
2. Next → Manual configuration → Python 3.10 → Next

### 2-4. WSGI 파일 수정
Web 탭에서 **WSGI configuration file** 링크 클릭
기존 내용 전체 삭제 후 아래 내용으로 교체:

```python
import sys
import os

project_home = '/home/Lifeff/isekai_market'
if project_home not in sys.path:
    sys.path = [project_home] + sys.path

os.chdir(project_home)

from app import app as application
```

저장(Save) 후 Web 탭으로 돌아가서 **Reload** 버튼 클릭

### 2-5. 접속 확인
```
https://Lifeff.pythonanywhere.com
```

## 3단계 - 코드 수정 후 업데이트하는 법

코드를 고칠 때마다 아래 순서로 하면 돼요.

**PyCharm 터미널에서:**
```bash
git add .
git commit -m "수정 내용 간단히 적기"
git push
```

**PythonAnywhere Bash 콘솔에서:**
```bash
cd /home/Lifeff/isekai_market
git pull
```

**Web 탭 → Reload 버튼 클릭 → 접속 확인**

## ✅ 로컬(내 PC)에서 테스트할 때

app.py 맨 아래 주석을 이렇게 바꾸세요:

```python
if __name__ == '__main__':
    app.run(debug=True)  # 이 줄 주석 해제
```

테스트 후 다시 push 전에 주석처리 되돌리기!
